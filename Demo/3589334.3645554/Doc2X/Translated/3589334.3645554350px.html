ABSTRACT
摘要

Zero-shot image classification, which aims to predict unseen classes whose samples have never appeared during the training phase, is crucial in the Web domain because many new web images appear on various websites. Attributes, as annotations for class-level characteristics, are widely used semantic information for this task. However, most current methods often fail to capture discriminative image features between similar images from different classes, leading to unsatisfactory zero-shot image classification results. This is because they solely focus on limited visual-attribute feature alignment. Therefore, we propose a Zero-Shot image Classification with Logic adapter and Rule prompt method called ZSCLR, which utilizes logic adapter and rule prompts to encourage the model to capture discriminative image features and achieve reasoning. Specifically, ZSCLR consists of a visual perception module and a logic adapter. The visual perception module extracts image features from training data. At the same time, the logic adapter utilizes the Markov logic network to encode the extracted image features and rule prompts for refining the discriminative image features. Due to predicates of rule prompts representing symbolic discriminative features, the proposed model can focus more on these discriminative features and achieve more precise image classification. Additionally, the logic adapter enables the model to adapt from recognizing images in seen classes to those in unseen classes through the reasoning of the Markov logic networks. We implement experiments on three standard zero-shot image classification benchmarks, and ZSCLR achieves competitive performance. Furthermore, ZSCLR can provide explanations for its predictions through rule prompts.

1 INTRODUCTION
1 引言

Relying on massive labeled training data to classify images has led to significant progress in the computer vision domain in recent years [23]. However, annotating all objects' classes is unrealistic, especially those that are expensive to label or not efficiently collectible in real-world applications. This limitation poses challenges for supervised learning methods when classifying objects that were unseen during training. To address this issue, zero-shot learning (ZSL) has emerged as a promising approach [15, 36]. ZSL requires labeled objects from seen classes but can effectively recognize objects from unseen classes. Applying ZSL to the computer vision domain is crucial because a vast number of new web images are created daily on social media and other websites.
依赖大量标记的训练数据对图像进行分类，在近年来计算机视觉领域取得了显著进展 [23]。然而，标注所有对象的类别是不切实际的，特别是那些标记成本高昂或在现实世界应用中无法有效收集的类别。这一限制给监督学习方法在分类训练期间未见过的对象时带来了挑战。为了解决这个问题，零样本学习（ZSL）作为一种有前景的方法应运而生 [15, 36]。ZSL 需要来自已见类别的标记对象，但可以有效地识别未见类别的对象。将 ZSL 应用于计算机视觉领域至关重要，因为每天在社交媒体和其他网站上都会产生大量新的网络图像。
In this paper, we focus on zero-shot image classification (ZSC), which involves training the model on images of seen categories and recognizing images of unseen categories. Most methods map images from the visual space into the semantic space and classify them [12,14,20,31] . Recently, several methods have emerged for learning discriminative image features in ZSC. Researchers have focused more on attention networks, leading to the development of attention-based ZSC methods [3,14,37] . These methods leverage attribute descriptions, e.g., word vectors, as auxiliary information to discover discriminative image features, facilitating accurate alignment with semantic representations. Although these efforts improve the classification accuracy in ZSC, the results are still unsatisfactory, particularly when handling datasets containing very visually similar images belonging to different classes. Because these approaches primarily rely on attention networks, which focus on limited semantic alignments between visual and attribute features. Logic rules, which condense human intelligence and knowledge, can be utilized to guide extracting discriminative image features. However, due to the challenge of integrating two distinct representations: logic rules in symbolic form and image representations in vector/matrix form, there is currently no existing method to apply them to ZSC tasks in terms of image classification from unseen classes. Therefore, in this paper, we seek to address the following question: How do we properly use logic rules to capture discriminative image features for improving zero-shot image classification accuracy?
最近，已经出现了几种在 ZSC 中学习判别性图像特征的方法。研究人员更多地关注注意力网络，导致了基于注意力的 ZSC 方法的开发 [3,14,37]。这些方法利用属性描述，例如词向量，作为辅助信息来发现判别性图像特征，促进与语义表示的准确对齐。尽管这些努力提高了 ZSC 中的分类精度，但结果仍然不尽如人意，特别是在处理包含属于不同类的非常视觉相似的图像的数据集时。因为这些方法主要依赖于注意力网络，这些网络专注于视觉和属性特征之间的有限语义对齐。逻辑规则，它们浓缩了人类的智慧和知识，可以被用来指导提取判别性图像特征。然而，由于将两种不同的表示形式：符号形式的逻辑规则和向量/矩阵形式的图像表示集成起来的挑战，目前还没有现有的方法将它们应用于从未见类别的图像分类的 ZSC 任务。因此，在本文中，我们试图解决以下问题：我们如何正确使用逻辑规则来捕获判别性图像特征以提高零样本图像分类的准确性？
To address the above issue, inspired by the human cognitive process, we will introduce a logic adapter for integrating logic rules to ZSC. Humans can comprehend unseen objects through reasoning based on prior experiences, even without prior exposure to them. For example, in Figure 1 (b), seen classes are "clay_colored_sparrow" with attribute features "cone_bill", "grey_wing" and "brewer_sparrow" with "dagger_bill", "brown_wing", while unseen class is "tree_sparrow". When humans encounter a new image of a "tree_sparrow" for the first time, they rely on prior knowledge from having seen images of "clay_colored_sparrow" and "brewer_sparrow". By comparing the new image's visual attributes, such as "cone_bill" and "brown_wing", to what they have learned from seen classes, humans can select the proper rule prompt,i.e.,cone_bill (x)∧ brown_wing (x) ⇒ tree_sparrow (x) ,from the knowledge base containing various rule prompts. Thus, they can logically deduce that the new image from the unseen class is indeed "tree_sparrow".
为了解决上述问题，受到人类认知过程的启发，我们将介绍一种逻辑适配器，用于将逻辑规则集成到ZSC中。人类可以通过基于先前经验的推理来理解未见过的物体，即使之前没有接触过它们。例如，在图1 (b)中，已见类别是“clay_colored_sparrow”，其属性特征为“cone_bill”、“grey_wing”和“brewer_sparrow”，其特征为“dagger_bill”、“brown_wing”，而未见类别是“tree_sparrow”。当人类第一次遇到“tree_sparrow”的新图像时，他们依赖之前看到“clay_colored_sparrow”和“brewer_sparrow”图像的先验知识。通过将新图像的视觉属性，如“cone_bill”和“brown_wing”，与他们从已见类别中学到的知识进行比较，人类可以从包含各种规则提示的知识库中选择适当的规则提示，即cone_bill (x)∧ brown_wing (x) ⇒ tree_sparrow (x)。因此，他们可以逻辑地推断出，来自未见类别的新图像确实是“tree_sparrow”。
Based on the above analysis, we have proposed a method called ZSCLR (Zero-Shot image Classification with Logic adapter and Rule prompt), which integrates rule prompts into ZSC models to capture discriminative image features. ZSCLR comprises two key components: a visual perception module and a logic adapter. The visual perception module, designed using CNN and attention network, primarily focuses on image feature extraction. Meanwhile, the logic adapter takes image features from the visual perception module and encodes them and rule prompts that predicates represent symbolic discriminative features via Markov logic network (MLN)[17]. More concretely, in the logic adapter, we introduce a statistical relation learning model, i.e.,MLN. MLN can combine statistical models (e.g., ZSC models) and relational models (e.g., rule prompts) to attain a unified representation, such as a joint probability distribution, and achieve reasoning via computing posterior. Moreover, rule prompts with symbolic discriminative features can tell the model which features are discriminative in an image, such as the shape of the bill, and then the model can focus more on what during training. In this paper, rule prompts are formalized through first-order logics (FOLs), such as dagger_bill (x)∧ brown_wing (x)⇒ brewer_sparrow (x) ,encoding them within the MLN. FOLs serve a dual purpose: they represent logical relationships between attribute features and classes and provide a powerful means of expressing symbolic knowledge. We use a variational Expectation-Maximization (EM) algorithm to train the model in an end-to-end way and utilize the logic adapter to reasoning results during the test. Additionally, the logic adapter offers a reasoning process through FOLs, which enables the model to adapt from seen classes to recognize unseen classes and explain why a particular decision was made. Finally, we present the results of our experiments conducted on the AwA2 [26], CUB [25] and SUN [16] datasets to evaluate the performance and interpretability of ZSCLR. These results vividly illustrate the remarkable superiority and promising potential of ZSCLR in zero-shot image classification.
基于以上分析，我们提出了一种名为ZSCLR（Zero-Shot图像分类与逻辑适配器及规则提示）的方法，该方法将规则提示整合到ZSC模型中，以捕捉区分性图像特征。ZSCLR包括两个关键组成部分：视觉感知模块和逻辑适配器。视觉感知模块采用CNN和注意力网络设计，主要关注图像特征提取。同时，逻辑适配器接收来自视觉感知模块的图像特征，并通过马尔可夫逻辑网络（MLN）[17]对它们和规则提示进行编码，规则提示通过谓词表示符号区分性特征。更具体地说，在逻辑适配器中，我们引入了一种统计关系学习模型，即MLN。MLN能够结合统计模型（例如ZSC模型）和关系模型（例如规则提示），以获得统一表示，如联合概率分布，并通过计算后验实现推理。此外，带有符号区分性特征的规则提示可以告诉模型图像中的哪些特征是区分性的，例如鸟喙的形状，然后模型在训练期间可以更多地关注这些特征。在本文中，规则提示通过一阶逻辑（FOLs）进行形式化，例如dagger_bill (x)∧ brown_wing (x)⇒ brewer_sparrow (x)，在MLN内部对它们进行编码。FOLs具有双重目的：它们表示属性特征与类别之间的逻辑关系，并提供了一种强大的表达符号知识的方式。我们使用变分期望最大化（EM）算法以端到端方式训练模型，并在测试期间利用逻辑适配器进行推理。此外，逻辑适配器通过FOLs提供了一个推理过程，使模型能够从已见类别适应以识别未见类别，并解释为什么做出特定的决策。最后，我们展示了在AwA2 [26]、CUB [25]和SUN [16]数据集上进行的实验结果，以评估ZSCLR的性能和可解释性。这些结果生动地展示了ZSCLR在零样本图像分类中的显著优越性和广阔的应用前景。

2 RELATED WORK
2 相关工作

Zero-shot learning aims to train a model to recognize classes not included in its original training. There are many works to study zero-shot learning in image classification. These works can be classified into three categories: embedding-based approaches, generative-based approaches and knowledge-based approaches. Embedding-based approaches aim to learn a mapping function for visual-semantic interaction. They determine the label of a sample by matching their vectors in the same space using similarity metrics. Some approaches are implemented by mapping the visual features to semantic space by [18,27,29] . In contrast,other methods propose mapping the semantic features into visual space and point out that using semantic space as shared latent space may reduce the variance of features [35]. As the embedding is learned only on seen classes, these embedding-based methods inevitably overfit to seen classes. To address this problem, generative-based methods have been introduced utilizing generative models such as VAEs, GANs to learn semantic-visual mapping to generate visual features of unseen classes, which can offset the shortage of unseen classes and convert ZSL into a supervised classification task. This category focuses on learning a class condition generator to generate features of unseen classes [1,33] or using semantic information (e.g.,attributions) of class to generate features of unseen classes directly [5,18,22] . However, these methods still usually yield relatively undesirable results since they cannot capture the subtle differences between seen and unseen classes. Knowledge-based methods have been explored to capture more correlations between seen and unseen classes. Most approaches utilize knowledge graphs (e.g., class hierarchies and commonsense knowledge) as side information. Specifically, they are used to build relationship graphs between seen and unseen classes as a semantics-level graph to learn recognizing unseen classes [2,10,11,24,30] .
零样本学习旨在训练一个模型，使其能够识别原始训练中未包含的类别。在图像分类中，有许多研究零样本学习的工作。这些工作可以分为三类：基于嵌入的方法、基于生成的方法和基于知识的方法。基于嵌入的方法旨在学习一个映射函数，用于视觉-语义交互。它们通过在相同空间中使用相似度度量匹配样本的向量来确定样本的标签。一些方法通过将视觉特征映射到语义空间来实现 [18,27,29] 。相比之下，其他方法提出将语义特征映射到视觉空间，并指出使用语义空间作为共享潜在空间可能会减少特征的方差 [35]。由于嵌入仅在已见类别上学习，这些基于嵌入的方法不可避免地会过度拟合到已见类别。为了解决这个问题，基于生成的方法被引入，利用生成模型如VAEs、GANs来学习语义-视觉映射，以生成未见类别的视觉特征，这可以弥补未见类别的不足，并将ZSL转化为监督分类任务。这一类别关注于学习一个类别条件生成器来生成未见类别的特征 [1,33] 或直接使用类别的语义信息（例如，属性）来生成未见类别的特征 [5,18,22] 。然而，这些方法通常仍然产生相对不理想的结果，因为它们无法捕捉到已见类别和未见类别之间的细微差异。基于知识的方法已被探索，以捕捉更多已见类别和未见类别之间的相关性。大多数方法利用知识图谱（例如，类别层次和常识知识）作为辅助信息。具体来说，它们被用来构建已见类别和未见类别之间的关系图，作为一个语义层面的图来学习识别未见类别 [2,10,11,24,30] 。
Besides the three mentioned categories, the zero-shot image classification methods can also be divided into discriminative and non-discriminative methods based on whether they consider the importance of different features. Since the above-mentioned approaches fail to account for discriminative features, they are attributed to non-discriminative methods. Recently, some discriminative methods have begun to explore the discriminative image features using attention networks [3,4,6,28,37] . Xie et al. [28] constructs a region graph using parts of the object via the attention technique and achieves transferring knowledge between different classes. Chen et al. [3,4] utilize mutually visual-attribute attention sub-net for semantic distillation, encouraging the model to explore the discriminative features for image. To solve the attribute imbalance and co-occurrence, Chen et al. [6] introduces an attribute-level contrastive learning mechanism.
除了上述提到的三类之外，零样本图像分类方法还可以根据是否考虑不同特征的重要性，分为判别式和非判别式方法。由于上述方法未能考虑判别式特征，它们被归类为非判别式方法。最近，一些判别式方法开始使用注意力网络 [3,4,6,28,37] 探索判别式图像特征。Xie等人 [28] 通过注意力技术使用对象的部件构建区域图，实现了不同类别之间的知识迁移。Chen等人 [3,4] 利用相互视觉属性注意力子网络进行语义蒸馏，鼓励模型探索图像的判别式特征。为了解决属性不平衡和共现问题，Chen等人 [6] 引入了属性级对比学习机制。
To some extent, our model is a discriminative approach. In contrast to existing approaches, we incorporate logic rules as auxiliary information to capture discriminative image features and logical relationships between discriminative features and classes, and employ the Markov logic network for prediction. Moreover, our model introduces rule prompts with symbolic discriminative features, providing interpretability compared to those who use attention networks to capture discriminative image features.
在某种程度上，我们的模型是一种判别式方法。与现有方法相比，我们结合逻辑规则作为辅助信息来捕捉判别式图像特征以及判别式特征与类别之间的逻辑关系，并使用马尔可夫逻辑网络进行预测。此外，我们的模型引入了带有符号判别式特征的规则提示，与使用注意力网络捕捉判别式图像特征的方法相比，提供了可解释性。

3.2 Logic Adapter
3.2 逻辑适配器

As the core of ZSCLR, the logic adapter (LA) integrates the VPM with rule prompts to extract discriminative image features. Meanwhile, LA can adapt the trained model from seen to unseen classes, and enables reasoning through Markov logic networks. To achieve this integration, it's crucial to establish a unified representation between the VPM and rule prompts. Thus, we introduce Markov logic networks (MLN), a framework that seamlessly combines statistical models (VPM) with relational models (rule prompt) into a unified representation [7]. For this purpose, we use FOLs like dagger_bill (x)∧ brown_wing (x)⇒ brewer_sparrow (x) as rule prompts. These FOLs establish logical relationships between symbolic discriminative features (attributes) and classes. Specifically, LA uses MLN to learn a joint probability distribution for symbolic discriminative features, enabling the prediction of discriminative feature labels by calculating posterior probabilities. The class labels are then inferred using fuzzy logic based on the predicted discriminative feature labels.
作为ZSCLR的核心，逻辑适配器（LA）将VPM与规则提示相结合，以提取区分性图像特征。同时，LA能够将训练好的模型从可见类适应到未见类，并通过马尔可夫逻辑网络进行推理。为了实现这种整合，关键在于在VPM和规则提示之间建立统一的表示。因此，我们引入了马尔可夫逻辑网络（MLN），一个能够无缝地将统计模型（VPM）与关系模型（规则提示）结合成统一表示的框架[7]。为此，我们使用如dagger_bill (x)∧ brown_wing (x)⇒ brewer_sparrow (x) 这样的FOL作为规则提示。这些FOL在符号区分性特征（属性）与类之间建立逻辑关系。具体来说，LA使用MLN学习符号区分性特征的联合概率分布，使得可以通过计算后验概率来预测区分性特征标签。然后，基于预测的区分性特征标签，使用模糊逻辑推断类标签。

3.4 Inference
3.4 推理

Prediction. After training ZSCLR, the model recognizes unseen images using the logic adapter. Specifically, the model uses the visual perception module to extract the discriminative image features, denoted as V′ . These discriminative image features are then input into the logic adapter. In the logic adapter, the feature network is used to calculate the posterior probability Qθ(V′) ,attaining the attribute feature labels of the instances. Then, the model leverages rule prompts from unseen classes to combine recognized attribute feature labels to infer class labels via fuzzy logic reasoning.
预测。训练 ZSCLR 后，模型使用逻辑适配器识别未见过的图像。具体来说，模型使用视觉感知模块提取区分性图像特征，记作 V′ 。这些区分性图像特征随后被输入到逻辑适配器中。在逻辑适配器中，特征网络用于计算后验概率 Qθ(V′) ，获得实例的属性特征标签。然后，模型利用未见类别的规则提示将识别的属性特征标签结合起来，通过模糊逻辑推理推断类别标签。
Interpretability. Our ZSCLR offers interpretability for predictions through corresponding rule prompts. That is to say, the model not only can predict class labels of objects in the images but also tell reasons for classifying the images to the class labels through symbolic language. In this study, we employ key attribute characteristics as our discriminative image features. To enhance understanding, we represent these attribute features using symbolic logic rules, replacing them in vector form. For example, when the model recognizes an unseen class image, it can identify the key attribute characteristics present in the sample and subsequently classify the unseen class object based on both these key attribute characteristics and the associated rule prompts. In this process, the key attribute characteristics and their corresponding rule prompts serve as explanations for the predictions.
可解释性。我们的 ZSCLR 通过相应的规则提示为预测提供可解释性。也就是说，模型不仅能预测图像中物体的类别标签，还能通过符号语言说明将图像分类到这些类别标签的原因。在本研究中，我们将关键属性特征作为区分性图像特征。为了增强理解，我们使用符号逻辑规则表示这些属性特征，将它们替换为向量形式。例如，当模型识别到一个未见类别的图像时，它可以识别样本中存在的关键属性特征，并随后基于这些关键属性特征和相关的规则提示对未见类别物体进行分类。在此过程中，关键属性特征及其相应的规则提示作为预测的解释。

Evaluation Protocols. We followed the evaluation protocol outlined in [26] and assessed the top-1 accuracy in two distinct settings. First, we conducted experiments within the conventional zero-shot learning (CZSL) framework, where exclusively unseen categories were involved in testing. Consequently, all prediction results were restricted to be drawn from unseen classes, and this accuracy is denoted as Acc. Secondly, we ventured into the realm of generalized zero-shot learning (GZSL), where test images encompassed both seen and unseen categories. In this setting, we computed the top-1 accuracy for both seen (S) and unseen (U) categories independently. Furthermore, to measure the trade-off between performance on seen and unseen categories, we calculated the harmonic mean (H) by using the formula H=2×S×US+U .
评估协议。我们遵循了文献 [26] 中概述的评估协议，并在两种不同的设置中评估了 top-1 准确率。首先，我们在传统的零样本学习（CZSL）框架内进行了实验，其中测试仅涉及未见过的类别。因此，所有的预测结果都限制在未见过的类别中，这种准确率表示为 Acc。其次，我们进入了广义零样本学习（GZSL）的领域，其中测试图像包含了见过和未见过类别。在这种设置中，我们分别独立地计算了见过（S）和未见（U）类别的 top-1 准确率。此外，为了衡量在见过和未见类别上性能的权衡，我们使用公式 H=2×S×US+U 计算了调和平均数（H）。
Implementation Details. We employed CNN as our basic image features extractor in the visual perception module. Indeed, our CNN can be initialized by a pre-trained backbone such as ResNet101 or GoogleNet [21]. Before being fed into the model, the images need to be randomly cropped for data augmentation. For optimization, we used the Adam optimizer with specific configurations. On the AwA2 dataset, we set the number of epochs to 15 , the batch size to 64 , and the learning rate to 1e-4. On the CUB dataset, we conducted training for 20 epochs, with a batch size of 32 and a learning rate of 1e-5. For SUN dataset, we set the number of epochs to 25 , the batch size to 64 , and the learning rate to 1 e-4 .
实施细节。我们在视觉感知模块中使用了 CNN 作为基本的图像特征提取器。实际上，我们的 CNN 可以通过诸如 ResNet101 或 GoogleNet [21] 的预训练主干网络进行初始化。在输入模型之前，图像需要被随机裁剪以进行数据增强。对于优化，我们使用了具有特定配置的 Adam 优化器。在 AwA2 数据集上，我们将训练周期数设为 15，批量大小设为 64，学习率设为 1e-4。在 CUB 数据集上，我们进行了 20 个周期的训练，批量大小为 32，学习率为 1e-5。对于 SUN 数据集，我们将训练周期数设为 25，批量大小设为 64，学习率设为 1 e-4。

5 CONCLUSION AND DISCUSSION
5 结论与讨论

In this paper, we propose the zero-shot image classification model with logic adapter and rule prompt (ZSCLR) to more accurately classify images in zero-shot scenes and provide the results' interpretability. ZSCLR consists of two modules: a visual perception module and a logic adapter. The visual perception module aims to extract image features. At the same time, the logic adapter takes image features from the visual perception module and encodes them and rule prompts via the Markov logic network. During training, the logic adapter refines these image features using backpropaga-tion to derive discriminative image features and adapt the model from seen classes to unseen classes. Furthermore, ZSCLR offers explanations for its prediction results through rule prompts with symbolic discriminative features. Comprehensive experiments conducted on three well-known benchmarks underscore the superior performance of ZSCLR. We believe that ZSCLR presents an innovative direction for the research community, particularly in the context of logic reasoning.
在本文中，我们提出了带有逻辑适配器和规则提示的零样本图像分类模型（ZSCLR），以更准确地对零样本场景中的图像进行分类，并提供结果的可解释性。ZSCLR由两个模块组成：视觉感知模块和逻辑适配器。视觉感知模块旨在提取图像特征。同时，逻辑适配器接收来自视觉感知模块的图像特征，并通过马尔可夫逻辑网络对它们和规则提示进行编码。在训练过程中，逻辑适配器通过反向传播精炼这些图像特征，以衍生出判别性图像特征，并将模型从已见类别适应到未见类别。此外，ZSCLR通过带有符号判别特征的规则提示为其预测结果提供解释。在三个知名基准上的综合实验凸显了ZSCLR的优越性能。我们相信ZSCLR为研究界，特别是在逻辑推理的背景下，展示了一个创新的方向。
Our ZSCLR not only predicts classes of images but also explains reasons for predictions by providing an interpretation in natural language form. Meantime, the model can locate the position of the discriminative features of objects. Therefore, in the future, ZSCLR has the potential to integrate into visual question models with natural language interactive Q&A. For instance, given an image and its prompt like "there are bill, wing and tail in the image" in the natural language style, the model can answer the following questions like "What is an object in this image?", "Why is the object?", "Where are the discriminative features for determining the object?", and so on.
我们的ZSCLR不仅能预测图像的类别，还能通过提供自然语言形式的解释来说明预测的原因。同时，该模型能够定位到物体判别性特征的位置。因此，在未来，ZSCLR有潜力整合到具有自然语言交互问答的视觉问题模型中。例如，给定一个图像及其自然语言风格的提示，如“图像中有账单、翅膀和尾巴”，模型可以回答以下问题，如“图像中的物体是什么？”，“为什么是这个物体？”，“确定物体的判别性特征在哪里？”等等。
