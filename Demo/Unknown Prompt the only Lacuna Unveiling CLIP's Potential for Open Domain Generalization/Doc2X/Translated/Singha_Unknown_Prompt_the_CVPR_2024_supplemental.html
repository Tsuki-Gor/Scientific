3. Extended literature survey of prompt learning using CLIP
3. 使用 CLIP 进行提示学习的扩展文献调研

Vision-Language Models (VLMs) have garnered significant interest across language processing and computer vision fields [3,5,16,25,37,43,44] . These models typically employ task-specific textual descriptions to interpret and analyze visual data [17,19] . While early prompting strategies relied on manual definitions,more recent developments have shifted towards automated prompt learning. CoOp [51] introduces an approach to optimize both unified and class-specific prompts via back-propagation. CoCoOp [50] further expands on CoOp by incorporating input-conditioned prompt learning, thus addressing issues related to generalization. The CLIP-adapter [15] innovates by fine-tuning feature adapters within both the visual and language branches of the model. ProGrad [54] is designed to prevent the forgetting of foundational knowledge within these models. TPT [41] leverages the consistency between multiple views of an image for supervision. Probabilistic and variational models such as Prod [28] and Varprompt [29] focus on learning prompt distributions that align with the spread of visual features. LASP [6] enhances the quality of learned prompts through a text-to-text cross-entropy loss. Meanwhile, MaPLe [21] works on improving the compatibility between different levels of CLIP encoders. However, a notable limitation of these approaches is their lack of specialization in handling multi-domain data, a crucial aspect for broader applicability in diverse real-world scenarios.
视觉-语言模型（VLMs）在语言处理和计算机视觉领域引起了广泛关注 [3,5,16,25,37,43,44] 。这些模型通常使用特定任务的文本描述来解释和分析视觉数据 [17,19] 。虽然早期的提示策略依赖于手动定义，但最近的进展已经转向自动化的提示学习。CoOp [51] 引入了一种优化统一和类别特定提示的方法，通过反向传播。CoCoOp [50] 在CoOp的基础上进一步扩展，通过整合输入条件化的提示学习，从而解决与泛化相关的问题。CLIP-adapter [15] 通过在模型的视觉和语言分支内微调特征适配器进行创新。ProGrad [54] 被设计用来防止这些模型遗忘基础知识。TPT [41] 利用图像多个视角之间的一致性进行监督。概率和变分模型，如Prod [28] 和Varprompt [29]，专注于学习与视觉特征分布一致的提示分布。LASP [6] 通过文本到文本的交叉熵损失增强了学习到的提示的质量。同时，MaPLe [21] 致力于提高不同级别的CLIP编码器之间的兼容性。然而，这些方法的一个显著局限是它们在处理多领域数据方面的专业性不足，这对于在多样化的现实世界场景中更广泛的应用至关重要。
In the realm of domain generalization, several researchers have investigated the concept of domain invariant prompts. For instance, [32] and [26] focus on harnessing text-based source domain knowledge or utilizing image patches as prompt inputs in Vision Transformer (ViT) models. This approach is akin to the methodology used in VPT [20], where prompts are adapted based on specific image features, aiming to achieve a more domain-agnostic model performance. DPL [49] employs CLIP [36] for multi-source Domain Generalization (DG) by deducing domain information from visual features on a batch-wise basis. However, DPL does not fully exploit CLIP's capability to discern domain-specific details. Additionally, it is prone to overfitting when dealing with small batches, as accurately estimating unbiased style characteristics becomes challenging.
在领域泛化的领域中，一些研究者探讨了领域不变提示的概念。例如，[32] 和 [26] 专注于利用基于文本的源领域知识，或者将图像块作为 Vision Transformer（ViT）模型的提示输入。这种方法类似于 VPT [20] 中使用的方法，其中提示基于特定图像特征进行适应，旨在实现更加领域无关的模型性能。DPL [49] 使用 CLIP [36] 进行多源领域泛化（DG），通过在批次基础上推导视觉特征中的领域信息。然而，DPL 并没有完全利用 CLIP 辨别领域特定细节的能力。此外，当处理小批次时，它容易过拟合，因为准确估计无偏风格特征变得具有挑战性。
As can be observed, our prompt learning technique stands out from all the previous literature.
如所见，我们的提示学习技术与之前的所有文献中的方法都不同。

4. Additional implementation details of the competitor models
4. 竞争对手模型的额外实现细节

In the CLIP+OpenMax configuration,we have developed a C+1 -class,threshold-free classifier using CLIP features to form a unified classifier. For the CLIP+OSDA variant, we incorporate a trainable linear layer on top of the pre-trained CLIP features, which acts as the generator. This is complemented by distinct discriminators for both source-specific classification and domain alignment. The adversarial aspect of this setup is implemented through a gradient-reversal layer, following the methodology outlined in [14].
在 CLIP+OpenMax 配置中，我们开发了一个 C+1 类别、无阈值的分类器，使用 CLIP 特征形成一个统一分类器。对于 CLIP+OSDA 变体，我们在预训练的 CLIP 特征之上加入了一个可训练的线性层，作为生成器。这由用于源特定分类和领域对齐的独特判别器补充。这种设置的对抗性方面通过遵循 [14] 中概述的方法，使用梯度反转层实现。
Regarding other prompt learning techniques, our implementation is faithful to the procedures described in the original works. For the CLIPN+STYLIP model, we divide the tokens into two separate categories. One category is shaped by the token learning strategy of STYLIP, and the other consists of specialized tokens that are modified in line with CLIPN's framework. This bifurcated token strategy effectively combines the strengths of both STYLIP and CLIPN, ensuring a harmonious and potent integration of these methodologies.
关于其他提示学习技术，我们的实现忠实于原作品中描述的程序。对于 CLIPN+STYLIP 模型，我们将标记划分为两个独立的类别。一个类别由 STYLIP 的标记学习策略形成，另一个类别包括根据 CLIPN 框架修改的专用标记。这种分支标记策略有效地结合了 STYLIP 和 CLIPN 的优点，确保了这些方法的和谐而强大的整合。
