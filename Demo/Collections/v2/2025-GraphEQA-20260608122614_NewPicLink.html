                                                                                                                                              Abstract: In Embodied Question Answering (EQA), agents must explore and develop a semantic understanding of an unseen environment to answer a situated question with confidence. This problem remains challenging in robotics, due to the difficulties in obtaining useful semantic representations, updating these representations online, and leveraging prior world knowledge for efficient planning and exploration. To address these limitations, we propose GraphEQA, a novel approach that utilizes real-time 3D metric-semantic scene graphs (3DSGs) and task relevant images as multi-modal memory for grounding Vision-Language Models (VLMs) to perform EQA tasks in unseen environments. We employ a hierarchical planning approach that exploits the hierarchical nature of 3DSGs for structured planning and semantics-guided exploration. We evaluate GraphEQA in simulation on two benchmark datasets, HM-EQA and OpenEQA, and demonstrate that it outperforms key baselines by completing EQA tasks with higher success rates and fewer planning steps. We further demonstrate GraphEQA in multiple real-world home and office environments. Videos and code: website. 摘要：在具身问答（EQA）中，智能体必须探索并建立对未知环境的语义理解，才能自信地回答情境问题。由于难以获得有用的语义表征、在线更新这些表征，以及为高效规划与探索利用先验世界知识，该问题在机器人领域仍然具有挑战性。为了解决这些限制，我们提出 GraphEQA，一种新的方法：它利用实时三维度量语义场景图（3DSGs）以及任务相关图像，作为多模态记忆，用于为视觉-语言模型（VLM）提供锚定，从而在未知环境中完成 EQA 任务。我们采用分层规划方法，借助 3DSGs 的分层特性实现结构化规划，并引导语义驱动的探索。我们在仿真中于两个基准数据集 HM-EQA 和 OpenEQA 上评估 GraphEQA，并表明它能以更高的成功率和更少的规划步骤完成 EQA 任务，从而优于关键基线。我们还在多个真实世界的家庭与办公环境中展示了 GraphEQA。视频与代码：网站。

                                                                                                                                              1 Introduction

                                                                                                                                              1 引言

                                                                                                                                              Embodied Question Answering (EQA) [1] is a challenging task in robotics, wherein an agent is required to explore and understand a previously unseen environment sufficiently well, to answer an embodied question in natural language. Accomplishing this task efficiently requires agents to rely on both commonsense knowledge of human environments as well as ground its exploration strategy in the current environment context. For example, to answer the question "How many chairs are there at the dining table?", the agent might rely on commonsense knowledge to understand that dining tables are often associated with dining rooms and dining rooms are usually near the kitchen towards the back of a home. A reasonable navigation strategy would involve navigating to the back of the house to locate a kitchen. To ground this search in the current environment, however, requires the agent to continually maintain an understanding of where it is, memory of where it has been, and what further exploratory actions will lead it to relevant regions. Finally, the agent needs to observe the target object(s) and perform visual grounding to reason about the number of chairs around the dining table, and confidently answer the question correctly. 具身问答（EQA）[1] 是机器人领域的一项具有挑战性的任务：智能体需要充分探索并理解一个先前未见的环境，才能用自然语言回答具身问题。高效完成该任务要求智能体既依赖关于人类环境的常识知识，又将其探索策略落到当前环境的上下文中。例如，对于问题“餐桌上有多少把椅子？”，智能体可能会利用常识来理解：餐桌通常对应餐厅，而餐厅通常靠近住宅后方的厨房。合理的导航策略是前往房屋后部以找到厨房。然而，要将这种搜索落到当前环境中，就要求智能体持续维持对自身位置的理解、对已走过区域的记忆，以及进一步探索行动将带它到达哪些相关区域。最后，智能体还需要观察目标物体，并进行视觉落地推理，以判断餐桌周围椅子的数量，并自信地正确回答问题。

                                                                                                                                              Maintaining a concise and effective memory and using it to take actions in the environment is critical for solving EQA tasks. Prior works have demonstrated the impressive commonsense reasoning capabilities of Vision Language Models (VLMs) as planning agents, while leveraging a semantic map for retrieval [2] or semantic exploration [3]. In such approaches, the VLMs are not grounded in the current environment, and commonsense reasoning and context-based decision-making are disconnected. Recent works [4,5,6,7,8] focus on maintaining memory modules that can be queried by VLM agents for grounded planning. To construct a semantically rich memory, prior works either maintain a large, extensive set of images [8, 9] or have to perform an expensive offline processing step to obtain a compact representation [6,7,10] . Thus,such semantic memory modules are either semantically rich [7,2,11] ,compact [4,5] ,or online [5] ,but not all at the same time. 保持简洁且有效的记忆，并用它来采取环境中的行动，对于解决 EQA 任务至关重要。已有工作表明，视觉语言模型（VLM）作为规划智能体具有令人印象深刻的常识推理能力，同时借助语义地图进行检索[2]或进行语义探索[3]。在这类方法中，VLM 并未扎根于当前环境，常识推理与基于上下文的决策是脱节的。近期工作 [4,5,6,7,8] 关注维护可被 VLM 智能体查询的记忆模块，用于扎根式规划。为构建语义丰富的记忆，既有工作要么维护一个庞大且冗长的图像集合[8, 9]，要么必须进行代价高昂的离线处理步骤以获得紧凑表示 [6,7,10] 。因此，这些语义记忆模块要么语义上丰富 [7,2,11] 、要么紧凑 [4,5] ，要么是在线 [5] ，但无法同时全部具备。

                                                                                                                                              To address these limitations, we propose GraphEQA, an approach for embodied question answering that builds an online, compact, multimodal semantic memory combining global, semantically-sparse, and task-agnostic information from real-time 3D scene graphs [12] with local and semantically-rich information from task-relevant images [10]. GraphEQA uses this multimodal representation for grounding vision-language planners to solve EQA tasks in unseen environments. Specifically, we utilize a recent spatial perception system [12] that incrementally creates a real-time 3D metric-semantic hierarchical scene graph (3DSG), given sequential egocentric image frames. We further augment this scene graph with semantic room labels and semantically-enriched frontiers, while maintaining a task-relevant visual memory that keeps track of task-relevant images as the robot explores the environment. Finally, we employ a hierarchical planning approach that utilizes the hierarchical nature of scene graphs and semantically relevant frontiers for structured planning and exploration in an unseen environment before using the multimodal memory to answer the embodied question with high confidence. 为了解决这些限制，我们提出 GraphEQA：一种具身问答方法，它构建在线、紧凑的多模态语义记忆，将来自实时三维场景图[12] 的全局、语义稀疏、且与任务无关的信息，与来自任务相关图像[10] 的局部、语义丰富信息相结合。GraphEQA 使用这种多模态表征来为视觉语言规划器提供视觉落地，从而在未见环境中解决 EQA 任务。具体而言，我们利用一种近期的空间感知系统[12]：在给定连续的以智能体视角（egocentric）图像帧的情况下，逐步生成实时三维度量-语义分层场景图（3DSG）。我们进一步为该场景图补充语义房间标签以及语义增强的前沿（frontiers），同时保持与任务相关的视觉记忆，用于在机器人探索环境时追踪与任务相关的图像。最后，我们采用一种分层规划方法：在使用多模态记忆回答具身问题并提高置信度之前，利用场景图的分层特性以及语义相关的前沿，在未见环境中进行结构化规划与探索。

                                                                                                                                              We demonstrate that given our real-time multimodal memory and hierarchical planning approach, the agent is able to accomplish long-horizon tasks with significantly fewer VLM planning steps, explores explainable task-relevant frontiers, and succeeds in EQA tasks with a higher rate than previous approaches. We demonstrate our results on the HM-EQA [3] and OpenEQA [13] datasets in the Habitat simulation environment [14] and also in the real world using the Hello Robot Stretch mobile manipulator in two scenes. While the focus of this work is EQA tasks, our contributions advance the broader goal of grounding vision-language foundation models in unseen 3D environments, making GraphEQA applicable to a wider range of robotics tasks. The key contributions of this work are as follows: 我们表明，借助我们的实时多模态记忆和分层规划方法，智能体能够以显著更少的 VLM 规划步骤完成长时程任务，探索可解释、与任务相关的前沿，并在 EQA 任务中以高于以往方法的成功率完成任务。我们在 Habitat 仿真环境 [14] 中的 HM-EQA [3] 和 OpenEQA [13] 数据集上展示了结果，也在现实世界中使用 Hello Robot Stretch 移动操作臂在两个场景中进行了验证。尽管本文重点是 EQA 任务，我们的贡献推动了将视觉-语言基础模型锚定到未见过的 3D 环境这一更广泛目标，使 GraphEQA 可适用于更大范围的机器人任务。本文的主要贡献如下：

                                                                                                                                              2 Related Work

                                                                                                                                              2 相关工作

                                                                                                                                              3D Semantic Scene Graph Representations for Planning: 3D semantic scene graphs (3DSGs) [15, 16, 17, 18, 19] have emerged as compact, semantically-rich representations of indoor environments, spurring advances in both offline [11, 2, 20, 21] and online [12, 22, 23] prediction. Offline approaches [11, 2] focus on enriching 3DSG nodes and edges with open-vocabulary semantics via Vision-Language Models (VLMs) [24, 25], supporting retrieval-based, language-guided tasks. Online methods [12, 22] enable real-time deployment of embodied agents but often rely on closed-set semantics [12] or fixed task sets [22], limiting open-world generalization. Our approach bridges these paradigms by building a multimodal memory: an online-constructed 3DSG with closed-set semantics guides a VLM agent toward task-relevant areas to capture semantically rich images [10], enabling open-world embodied question answering (EQA). Recent work has leveraged textual 3DSG representations for VLM-based planning [4, 26, 27, 28], primarily targeting object search or rearrangement tasks, but without addressing the deeper semantic understanding required for EQA. Other spatio-temporal representations of environments have also been effective for planning, e.g., [29], which constructs a custom 3D data structure to maintain dynamic memory. 用于规划的 3D 语义场景图表征：3D 语义场景图（3DSGs）[15, 16, 17, 18, 19] 已成为紧凑且语义丰富的室内环境表征，推动了离线 [11, 2, 20, 21] 与在线 [12, 22, 23] 预测的进展。离线方法 [11, 2] 通过视觉-语言模型（VLMs）[24, 25]，用开放词汇语义来丰富 3DSG 的节点与边，从而支持基于检索、语言引导的任务。在线方法 [12, 22] 使具身智能体能够实时部署，但往往依赖封闭集合语义 [12] 或固定任务集 [22]，从而限制开放世界泛化。我们的方案通过构建多模态记忆来衔接上述范式：先在线构建一个带封闭集合语义的 3DSG，引导 VLM 智能体定位任务相关区域，以捕获语义丰富的图像 [10]，从而实现开放世界具身问答（EQA）。近期工作还利用文本形式的 3DSG 表征进行 VLM 驱动的规划 [4, 26, 27, 28]，主要面向物体搜索或重排任务，但尚未解决 EQA 所需的更深层语义理解。其他时空环境表征也对规划有效，例如 [29] 通过构建自定义 3D 数据结构来维护动态记忆。

                                                                                                                                              VLMs for 3D Scene Understanding and Planning: Several previous works leverage the commonsense reasoning capabilities of foundation models for long-horizon planning [30, 31, 32]. However, these methods are not grounded in the context of the current environment and additional tools are required to translate the LLM plan to executable actions [33, 34, 31, 35]. Previous works have explored the use of VLMs for building dense queryable open-vocabulary 3D semantic representations using explicit pixel-level [36, 37, 38, 39, 40] or implicit neural [41, 42, 43] representations. However, these maps are built offline, before they are used for downstream retrieval-based tasks. Moreover, such dense representations are not scalable to large environments and cannot be used to ground VLM-based planners. Recent advancements in grounding VLM-based planners using videos [44, 8] are promising, but struggle with scalability for long-horizon tasks in large environments. VLMs have been used as planners while leveraging semantic maps for retrieval [45,2,11] or semantic exploration [3,46,47,48,49] ,however such approaches disconnect context-based decision-making and commonsense reasoning. Offline methods that build topological maps [50], keyframe selections [10], 3D semantic graphs [2, 11, 51, 20, 21] and experience summaries [6, 7, 52, 53] are unsuitable for real-time deployment in novel settings. Online semantic scene graphs, while real-time, are limited by closed-set semantics. Our approach introduces an online, compact, and semantically rich multimodal memory to effectively ground VLM planners for EQA tasks. 用于 3D 场景理解与规划的 VLM：已有多项工作利用基础模型的常识推理能力来进行长时域规划 [30, 31, 32]。然而，这些方法并未基于当前环境语境，并且需要额外工具将 LLM 计划转换为可执行动作 [33, 34, 31, 35]。先前工作还探讨了使用 VLM 构建密集可查询的开放词汇 3D 语义表征，方式包括显式像素级 [36, 37, 38, 39, 40] 或隐式神经 [41, 42, 43] 表征。但这些图在用于下游基于检索任务之前，是离线构建的。此外，这类密集表征难以扩展到大规模环境，且无法用于为 VLM 驱动的规划器提供落地点。利用视频对 VLM 驱动规划器进行落地（grounding）的最新进展 [44, 8] 很有前景，但在大规模环境中的长时域任务上仍面临可扩展性挑战。VLM 被用作规划器，同时利用语义地图进行检索 [45,2,11] 或语义探索 [3,46,47,48,49] ，然而这类方法会割裂基于语境的决策与常识推理。构建拓扑图 [50]、关键帧选择 [10]、3D 语义图 [2, 11, 51, 20, 21] 以及经验摘要 [6, 7, 52, 53] 的离线方法，不适用于新环境中的实时部署。尽管在线语义场景图具有实时性，但仍受限于封闭集合语义。我们的方案提出一种在线、紧凑且语义丰富的多模态记忆，用于有效落地 VLM 规划器以完成 EQA 任务。

                                                                                                                                              Embodied Question Answering: Embodied Question Answering [1, 54, 55, 56] has emerged as a challenging paradigm for testing robotic task planning systems on their ability to incrementally build a semantic understanding of an environment in order to correctly answer an embodied question with confidence. Ren et al. [3] build an explicit task-specific 2D semantic map of the environment to guide exploration, Anwar et al. [6], Xie et al. [7] build offline experience modules that the LLM can query, and Majumdar et al.                                                                                                                                               3 方法

                                                                                                                                              3.1 Preliminaries

                                                                                                                                              3.1 预备知识

                                                                                                                                              Hierarchical 3D Metric-semantic Scene Graphs. 3D metric-semantic scene graphs (3DSGs) provide a structured, layered representation of environments and encode spatial, semantic, and relational information [15, 16, 17]. Recent works like Clio [22], Hydra [12], and Open Scene Graphs [57] introduce efficient real-time frameworks for incremental construction of hierarchical SG layers consisting of objects, regions, rooms, buildings, etc. See Appendix A. 3 for details. 层次化3D度量-语义场景图。3D度量-语义场景图（3DSG）为环境提供一种结构化、分层的表示，并编码空间、语义和关系信息[15, 16, 17]。Clio[22]、Hydra[12]和Open Scene Graphs[57]等近期工作提出了高效的实时框架，可增量构建由物体、区域、房间、建筑等组成的层次化SG层。详见附录A.3。

                                                                                                                                              2D Occupancy Mapping and Frontier Detection. 3D voxel-based occupancy maps are an effective way for storing explored, occupied, and unexplored regions of an environment for planning and navigation. As the robot explores, using depth data and camera intrinsics, occupancy of the voxels is updated using Volumetric Truncated Signed Distance Function (TSDF) fusion. TSDF integrates depth observations to update voxels as occupied or free, while areas beyond a certain threshold are marked unexplored. Typically, the 3D occupancy map is projected into 2D, where frontiers-boundaries between explored and unexplored regions—are identified to guide further exploration. We employ this approach in our method for identifying frontiers, clustering them and adding them to the scene graph. 二维占据映射与边界检测。基于3D体素的占据地图是存储环境中已探索、已占据和未探索区域的有效方式，可用于规划和导航。机器人在探索时，利用深度数据和相机内参，通过体积截断签名距离函数（TSDF）融合更新体素占据情况。TSDF将深度观测融合起来，把体素更新为已占据或空闲，而超过某一阈值的区域则标记为未探索。通常，3D占据地图会投影到二维，其中边界——已探索与未探索区域之间的边界——会被识别出来，以引导进一步探索。我们在方法中采用这一思路来识别边界、对其聚类，并将其加入场景图。

As the agent explores the environment, it used its sensor data (RGBD images, semantic map, camera poses and intrinsics) to construct a 3D metric-semantic hierarchical scene graph (3DSG) as well as a 2D occupancy map for frontier selection in real time. The constructed 3DSG is enriched as discussed in Section 3.3. From the set of images collected during each trajectory execution, a task-relevant subset is selected, called the task-relevant visual memory ( Section 3.4).                                                                                                                                               3.2 Problem Formulation

                                                                                                                                              3.2 问题表述

                                                                                                                                              An overview of our proposed method is shown in Figure 2. Given a multiple choice question q ,we wish to find the correct answer a∗∈A where A is the set of multiple-choice answers to question q available to the agent. To find a∗ ,the agent is equipped with a VLM-based planner V(q,Ste,{Ik}k=1K,Ht,Xt)=(at,ct,ut) ,where Ste is the enriched scene graph constructed online at planning time t ,and includes frontier nodes from 2D the occupancy map (Section 3.1). {Ik}k=1K is a small set of task-relevant images maintained in memory (Section 3.4), at is the current best answer to the multiple choice question q and ut is the next action the agent should take in the environment. Ht represents the accrued history up to time t and Xt represents the current state of the agent. We query the VLM planner V (Section 3.5) at time t with the inputs described above, with the scene graph Ste continually being constructed and a set of images {Ik}k=1K chosen based on task and semantic relevance. The planner then outputs a high-level action ut which is executed in the environment while the scene graph, visual memory, and frontiers are all updated in real time.
