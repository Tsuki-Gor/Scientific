Abstract
摘要

The Zero-Shot Object Navigation (ZSON) task requires embodied agents to find a previously unseen object by navigating in unfamiliar environments. Such a goal-oriented exploration heavily relies on the ability to perceive, understand, and reason based on the spatial information of the environment. However, current LLM-based approaches convert visual observations to language descriptions and reason in the linguistic space, leading to the loss of spatial information. In this paper, we introduce TopV-Nav, a MLLM-based method that directly reasons on the top-view map with complete spatial information. To fully unlock the MLLM's spatial reasoning potential in top-view perspective, we propose the Adaptive Visual Prompt Generation (AVPG) method to adaptively construct semantically-rich top-view map. It enables the agent to directly utilize spatial information contained in the top-view map to conduct thorough reasoning. Besides, we design a Dynamic Map Scaling (DMS) mechanism to dynamically zoom top-view map at preferred scales, enhancing local fine-grained reasoning. Additionally, we devise a Target-Guided Navigation (TGN) mechanism to predict and to utilize target locations, facilitating global and human-like exploration. Experiments on MP3D and HM3D benchmarks demonstrate the superiority of our TopV-Nav,e.g., +3.9% SR and +2.0% SPL absolute improvements on HM3D.
零样本物体导航（ZSON）任务要求具身代理在不熟悉的环境中找到之前未见过的物体。这种目标导向的探索在很大程度上依赖于感知、理解和基于环境空间信息进行推理的能力。然而，目前基于 LLM 的方法将视觉观察转换为语言描述，并在语言空间中进行推理，导致空间信息的丧失。在本文中，我们介绍了 TopV-Nav，一种基于 MLLM 的方法，它直接在具有完整空间信息的顶视地图上进行推理。为了充分释放 MLLM 在顶视视角下的空间推理潜力，我们提出了自适应视觉提示生成（AVPG）方法，以自适应地构建语义丰富的顶视地图。它使代理能够直接利用顶视地图中包含的空间信息进行深入推理。此外，我们设计了一种动态地图缩放（DMS）机制，以在首选比例下动态缩放顶视地图，增强局部细粒度推理。此外，我们还设计了一种目标引导导航（TGN）机制，以预测和利用目标位置，促进全球和类人探索。在 MP3D 和 HM3D 基准上的实验表明我们的 TopV-Nav 的优越性，例如， +3.9% SR 和 +2.0% SPL 在 HM3D 上的绝对提升。

1. Introduction
1. 引言

In the realm of embodied AI, Zero-Shot Object Navigation (ZSON) is a fundamental challenge, requiring an agent to traverse in continuous environments to locate a previously-unseen object specified by category (e.g., fireplace). Such a zero-shot setting discards category-specific training and supports an open-category manner, emphasizing reasoning and exploration ability with minimized movement costs.
在具身人工智能领域，零-shot物体导航（ZSON）是一个基本挑战，要求代理在连续环境中移动，以定位由类别指定的先前未见物体（例如，壁炉）。这种零-shot设置放弃了特定类别的训练，支持开放类别的方式，强调推理和探索能力，同时最小化移动成本。
Recently, emerging works [34, 48, 49] have started to integrate Large Language Models (LLMs) into ZSON agents, aiming to improve the reasoning ability by harnessing the extensive knowledge embedded in LLMs. For instance, objects co-occurrence knowledge, e.g., a mouse often being found near a computer, can guide agent's exploration.
最近，出现的研究 [34, 48, 49] 开始将大型语言模型（LLMs）整合到ZSON代理中，旨在通过利用LLMs中嵌入的广泛知识来提高推理能力。例如，物体共现知识，例如鼠标通常在计算机附近，可以指导代理的探索。

As is well known, spatial information is vital for navigation decision-making, as it includes essential aspects such as room layouts and positional relationships among objects. Typically, navigation agents translate egocentric observations into a structured map to encode spatial information, i.e., top-view map or called bird's eye view map. This map serves as the core representation, facilitating essential functionalities like obstacle avoidance and path planning.
众所周知，空间信息对导航决策至关重要，因为它包含房间布局和物体之间位置关系等基本方面。通常，导航代理将自我中心的观察转换为结构化地图，以编码空间信息，即顶视图地图或称为鸟瞰图。这张地图作为核心表示，促进了障碍物避免和路径规划等基本功能。
However, current methods face notable limitations. As illustrated in Fig.1(a), these methods lie in two paradigms, i.e., frontier-based exploration (FBE) or waypoint-based exploration methods. Typically, LLM-based methods need to convert the top-view map into natural language, e.g., surrounding descriptions, and use LLM to conduct reasoning in the linguistic domain. This map-to-text conversion process leads to the loss of vital spatial information such as the layout information of the living room. Alternatively, if the agent knows the spatial layout of the living room and understands that the fireplace is generally positioned opposite the sofa and table, the spatial location of the fireplace can be directly inferred based on the room layout. Therefore, considering the top-view map contains useful spatial information, and that MLLMs have demonstrated capabilities in grasping spatial relationships within images in the field of image understanding [5,25] ,an interesting question arises "can we leverage MLLM to reason directly on the image of top-view map to produce executable moving decisions?"
然而，目前的方法面临显著的局限性。如图1(a)所示，这些方法分为两种范式，即基于边界的探索（FBE）或基于航点的探索方法。通常，基于大型语言模型（LLM）的方法需要将俯视图地图转换为自然语言，例如周围环境描述，并利用LLM在语言领域进行推理。这一地图到文本的转换过程导致了重要空间信息的丢失，例如客厅的布局信息。另一方面，如果代理知道客厅的空间布局，并理解壁炉通常位于沙发和桌子的对面，则可以根据房间布局直接推断出壁炉的空间位置。因此，考虑到俯视图地图包含有用的空间信息，并且多模态大型语言模型（MLLM）在图像理解领域展示了把握图像中空间关系的能力 [5,25]，一个有趣的问题出现了：“我们能否利用MLLM直接在俯视图地图的图像上进行推理，以产生可执行的移动决策？”

Besides, as shown in Fig.1(a), FBE methods select a point from frontier regions to move toward exploration, restricting the LLM's action space to only the frontier boundaries. Waypoint-based methods use the waypoint predictor to generate navigable waypoints and select a waypoint to move toward exploration, restricting the LLM's action space to only a predefined set of points. Moreover, since the waypoint predictor is trained offline using depth information, its predicted waypoints focus solely on traversabil-ity without semantics, also leading to weakly-semantic action space. Both FBE and waypoint-based paradigms suffer from constrained local action spaces. Thus, the question is "can we construct global and semantic-rich action space?"
此外，如图1(a)所示，FBE方法从边界区域选择一个点以进行探索，限制了LLM的行动空间仅限于边界。基于航点的方法使用航点预测器生成可导航的航点，并选择一个航点进行探索，限制了LLM的行动空间仅限于预定义的一组点。此外，由于航点预测器是使用深度信息离线训练的，其预测的航点仅关注可通行性而不考虑语义，这也导致了弱语义的行动空间。FBE和基于航点的范式都面临局部行动空间受限的问题。因此，问题是：“我们能否构建全球和富含语义的行动空间？”
Furthermore, when humans explore unfamiliar environments, they can use current observations to infer the layout of unseen areas and predict where the target object might be located [15]. This predicted target location guides their movement decisions strategically, even in the absence of complete information. However, the action spaces of FBE and waypoint-based method are confined strictly to navigable regions, which prevents them from possessing this capability. Thus, the question is "can we leverage known observations to infer the probable location of the target object in unexplored areas to guide the current decision?"
此外，当人类探索不熟悉的环境时，他们可以利用当前的观察来推断未见区域的布局，并预测目标物体可能的位置 [15]。这个预测的目标位置在缺乏完整信息的情况下，战略性地指导他们的移动决策。然而，FBE 和基于航点的方法的行动空间严格限制在可导航区域内，这使得它们无法具备这种能力。因此，问题是“我们能否利用已知的观察来推断目标物体在未探索区域的可能位置，以指导当前的决策？”
Therefore, to address the limitations and questions mentioned above, we make multi-fold innovations. First, we propose an insightful method called TopV-Nav to fully unlock the top-view spatial reasoning potential of MLLM for ZSON task. Specifically, the current LLM-driven paradigm requires the map-to-language process for LLM reasoning in linguistic space, where the converting process may lose some crucial spatial information such as objects and room layout. Instead, we propose a novel paradigm that leverages MLLM to directly reason on the top-view map, discarding the map-to-language process and maximizing the utilization of all the spatial information. Second, we introduce an Adaptive Visual Prompt Generation (AVPG) method to enhance MLLM's understanding of the top-view map. AVPG adaptively generates visual prompts directly onto the map, in which various elements are spatially arranged to reflect their spatial relationships within the environment. Therefore, MLLM can grasp and comprehend crucial information directly from the map, facilitating effective spatial reasoning. For instance, in Fig.1(b), our method can interpret the room's layout and infer the fireplace's location. Additionally, the moving location is predicted directly based on the top-view map, resulting in a global and more flexible action space. Third, for environments with numerous objects, the top-view map may not be able to visually represent all elements. Thus, we propose a Dynamic Map Scaling (DMS) mechanism to dynamically adjust the map's scale via zooming operations. DMS further enhances agent's local spatial reasoning and fine-grained exploration in local regions. Last but not least, we propose a Target-Guided Navigation (TGN) mechanism to first predict the final coordinate of the target object, which can even lie in unexplored areas. The target coordinate further guides the moving location within current navigable regions, mirroring human-like predictive reasoning and exploratory behavior.
因此，为了解决上述提到的局限性和问题，我们进行了多方面的创新。首先，我们提出了一种名为 TopV-Nav 的深刻方法，以充分释放 MLLM 在 ZSON 任务中的顶视空间推理潜力。具体而言，当前的 LLM 驱动范式需要将地图转化为语言，以便在语言空间中进行 LLM 推理，而转换过程可能会丢失一些关键的空间信息，例如物体和房间布局。相反，我们提出了一种新颖的范式，利用 MLLM 直接在顶视地图上进行推理，舍弃地图到语言的过程，最大限度地利用所有空间信息。其次，我们引入了一种自适应视觉提示生成（AVPG）方法，以增强 MLLM 对顶视地图的理解。AVPG 自适应地直接在地图上生成视觉提示，其中各种元素在空间上排列，以反映它们在环境中的空间关系。因此，MLLM 可以直接从地图中掌握和理解关键的信息，从而促进有效的空间推理。例如，在图 1(b) 中，我们的方法可以解释房间的布局并推断壁炉的位置。此外，移动位置是直接基于顶视地图预测的，从而形成一个全球性和更灵活的行动空间。第三，对于具有众多物体的环境，顶视地图可能无法视觉上表示所有元素。因此，我们提出了一种动态地图缩放（DMS）机制，通过缩放操作动态调整地图的比例。DMS 进一步增强了代理在局部区域的空间推理和细粒度探索。最后但并非最不重要的是，我们提出了一种目标引导导航（TGN）机制，首先预测目标物体的最终坐标，该坐标甚至可能位于未探索的区域。目标坐标进一步引导当前可导航区域内的移动位置，反映出类似人类的预测推理和探索行为。
Experiments are conducted on MP3D and HM3D benchmarks, which demonstrates that our TopV-Nav achieves promising performance,e.g., +3.9% SR and +2.0% SPL absolute improvements on HM3D.
实验在 MP3D 和 HM3D 基准上进行，表明我们的 TopV-Nav 在性能上取得了令人鼓舞的成果，例如，在 HM3D 上的 +3.9% SR 和 +2.0% SPL 的绝对提升。

2.1. Object-goal Navigation
2.1. 目标导向导航

Object-goal navigation has been a fundamental challenge in embodied AI [4, 6, 11-13, 22, 23, 28, 29, 38-40, 44- 47,50]. Early methods leverage RL to train policies, which explore visual representations [24], meta-learning [33], and semantic priors [35,37] etc.,to enhance performance. Modular-based approaches [3, 27, 43] leverage perception models [14, 42] to construct episodic maps, based on which long-term goals are generated to guide the local policy.
目标导向导航一直是具身人工智能中的一个基本挑战 [4, 6, 11-13, 22, 23, 28, 29, 38-40, 44-47, 50]。早期的方法利用强化学习来训练策略，探索视觉表示 [24]、元学习 [33] 和语义先验 [35,37] 等，以提高性能。基于模块的方法 [3, 27, 43] 利用感知模型 [14, 42] 构建情景地图，基于此生成长期目标以指导局部策略。
To overcome closed-world assumption and achieve zero-shot object navigation (ZSON) task, EmbCLIP [16] and [21] leverage the multi-modal alignment ability of CLIP [26] to enable cross-domain zero-shot object navigation. Furthermore, CoWs [10] accelerates the progress of the ZSON task, where no simulation training is required, and a single model can be applied across multiple environments. Recent methods [34, 49] extract semantic information using powerful off-the-shelf detectors [19, 20, 41], based on which they employ LLMs to determine the next frontier [49] or waypoint [34] for exploration. However, spatial layout information is lost during map-to-language conversion. To address this limitation, we propose direct reasoning on the top-view map with MLLM, fully leveraging the complete spatial information.
为了克服封闭世界假设并实现零-shot 目标导航 (ZSON) 任务，EmbCLIP [16] 和 [21] 利用 CLIP [26] 的多模态对齐能力来实现跨领域的零-shot 目标导航。此外，CoWs [10] 加速了 ZSON 任务的进展，其中不需要模拟训练，单个模型可以应用于多个环境。最近的方法 [34, 49] 使用强大的现成检测器 [19, 20, 41] 提取语义信息，基于此它们利用 LLMs 确定下一个探索前沿 [49] 或航点 [34]。然而，在地图到语言的转换过程中，空间布局信息丢失。为了解决这一限制，我们提出在顶视图地图上进行直接推理，利用 MLLM，充分利用完整的空间信息。

2.2. Spatial Reasoning with MLLMs
2.2. 使用 MLLM 的空间推理

Developing the spatial reasoning capabilities of MLLMs has become increasingly popular recently. KAGI [17] proposes generating a coarse robot movement trajectory as dense reward supervision through keypoint inference. SCAFFOLD [18] leverages scaffolding coordinates to promote vision-language coordination. PIVOT [25] iteratively prompts MLLM with images annotated with a visual representation of proposals and can be applied to a wide range of embodied tasks. In the domain of vision-language navigation, AO-Planner [7] proposes visual affordance prompting to enable MLLM to select candidate waypoints from front-view images. However, previous works focus on exploring MLLM's spatial reasoning from egocentric perspectives, while the investigation from top-view perspective remains limited (top-view map is the core representation for robots). Our work explores the top-view spatial reasoning potential of MLLMs for the ZSON task.
最近，发展 MLLM 的空间推理能力变得越来越受欢迎。KAGI [17] 提出通过关键点推断生成粗略的机器人运动轨迹作为密集奖励监督。SCAFFOLD [18] 利用支架坐标来促进视觉-语言协调。PIVOT [25] 通过带有提案视觉表示的图像迭代提示 MLLM，并可应用于广泛的具身任务。在视觉-语言导航领域，AO-Planner [7] 提出了视觉可供性提示，使 MLLM 能够从前视图图像中选择候选路径点。然而，以前的工作主要集中在从自我中心视角探索 MLLM 的空间推理，而从顶视角进行的研究仍然有限（顶视图地图是机器人核心表示）。我们的工作探索 MLLM 在 ZSON 任务中的顶视空间推理潜力。

3. Method
3. 方法

3.1. Problem Definition
3.1. 问题定义

The ZSON task requires an agent, which is randomly placed in a continuous environment as initialization, to navigate to an instance of a user-specified object category G . At time step t ,the agent receives egocentric observations Ot ,which contains RGB-D images and its pose pt . The agent is expected to adopt a low-level action at ,i.e.,move_forward, turn_left, turn_right, look_up, look_down and stop. The task is considered successful if the agent stops within a distance threshold from the target in less than 500 steps and the target is visible in the egocentric observation.
ZSON 任务要求一个代理在一个连续环境中随机初始化，并导航到用户指定的对象类别实例 G 。在时间步 t ，代理接收自我中心观察 Ot ，其中包含 RGB-D 图像及其姿态 pt 。代理预计采取低级动作 at ，即 move_forward、turn_left、turn_right、look_up、look_down 和 stop。如果代理在少于 500 步内停在目标的距离阈值内，并且目标在自我中心观察中可见，则任务被视为成功。

3.2. Overview
3.2. 概述

We propose a novel MLLM-driven method, termed TopV-Nav, designed to fully unlock MLLM's top-view perception and spatial reasoning capabilities for the ZSON task.
我们提出了一种新颖的 MLLM 驱动方法，称为 TopV-Nav，旨在充分释放 MLLM 在 ZSON 任务中的顶视感知和空间推理能力。
Specifically, as shown in Fig.2, the agent observes egocentric RGB-D images It and its pose pt at time step t . Then,the proposed Adaptive Visual Prompt Generation (AVPG) module converts these egocentric observations onto a top-view map VMt as shown in (a). Note that VMt adopts different colors and text-boxes as visual prompts to represent various elements, which are spatially arranged on the map to provide spatial layout information. Next, in the proposed Dynamic Map Scaling (DMS) module, we take VMt to query MLLM to conduct spatial reasoning and predict a ratio r∈[0,1] for zooming in the map as shown in (b)(c). Note that the zooming region is the agent's surroundings, supporting local fine-grained exploration. Subsequently, we propose Target-Guided Navigation (TGN) as shown in Fig.2. The scaled map is taken as input to query MLLM to conduct spatial reasoning and produce the target location and moving location. The target location refers to the potential location of the target object, thus it may lie in unknown areas on the map. This target location is used to guide the prediction of the moving location, which is the agent's next actual destination within navigable areas. Note that any location within navigable areas can be decided as the moving location, resulting in a global action space that is not restricted to frontiers or waypoints. Finally, we utilize the fast marching method [31] as the local policy to perform a series of low-level actions, by which the agent can gradually move towards the moving location.
具体而言，如图2所示，代理观察自我中心的RGB-D图像It及其在时间步t的姿态pt。然后，所提出的自适应视觉提示生成（AVPG）模块将这些自我中心的观察转换为如(a)所示的顶视图地图VMt。请注意，VMt采用不同的颜色和文本框作为视觉提示，以表示各种元素，这些元素在地图上空间排列，以提供空间布局信息。接下来，在所提出的动态地图缩放（DMS）模块中，我们使用VMt查询MLLM进行空间推理，并预测缩放地图的比例r∈[0,1]，如(b)(c)所示。请注意，缩放区域是代理的周围环境，支持局部细粒度探索。随后，我们提出了目标引导导航（TGN），如图2所示。缩放后的地图作为输入查询MLLM进行空间推理，并生成目标位置和移动位置。目标位置指的是目标物体的潜在位置，因此它可能位于地图上的未知区域。该目标位置用于指导移动位置的预测，移动位置是代理在可导航区域内的下一个实际目的地。请注意，在可导航区域内的任何位置都可以被决定为移动位置，从而形成一个不受边界或航点限制的全局动作空间。最后，我们利用快速行进方法[31]作为局部策略，执行一系列低级动作，使代理能够逐渐朝向移动位置移动。

3.3. Adaptive Visual Prompt Generation
3.3. 自适应视觉提示生成

To adaptively construct top-view maps on which MLLM can perform spatial layout reasoning, we propose the Adaptive Visual Prompt Generation (AVPG) method.
为了自适应地构建顶视图地图，使得MLLM能够进行空间布局推理，我们提出了自适应视觉提示生成（AVPG）方法。
Technically,at each time step t ,we transform the egocentric depth image dt into 3D point clouds utilizing agent’s pose pt and then convert them to the global space. Points near the floor are seen to the navigable areas while points above a height threshold hθ are considered obstacle areas. Moreover, we leverage Grounding-DINO [20] to identify objects and their bounding boxes from egocentric images. Each box is projected into 3D using corresponding depth information and further projected to the top-view map by compressing high dimensions. To build a map that MLLM can understand, we leverage different colors and text as visual prompts on the map to distinguish different areas, such as historical traveled areas, navigable areas, obstacles areas, frontiers and objects, as shown in Fig.2(a). For detected objects, we devise a simple but efficient sampling strategy to select goal-relevant objects to display on the map. We assign each bounding box j of category i a priority score scorei,j which is formulated as:

Technically, we apply DBSCAN [9], a density-based spatial clustering algorithm. We cluster two types of elements: frontiers and the locations of observed objects. In one respect, by identifying the boundary points of the navigable areas and the locations of obstacles, we obtain the locations of frontiers not occupied by obstacles ⋃i∈F⁡(xi,yi) . In another respect,we leverage detection model and depth information to collect observed objects' locations ⋃i∈O⁡(xi,yi) . Eventually,we merge these two parts into a candidate point set ⋃i∈O∪F⁡(xi,yi) and apply DBSCAN method to it.

3.4. Dynamic Map Scaling
3.4. 动态地图缩放

The top-view map VMt with fixed size and resolution may not effectively represent the spatial layout. Therefore, we devise the Dynamic Map Scaling (DMS) mechanism to dynamically adjust the map scale during exploration, further supporting local fine-grained exploration. For instance, as shown in the original top-view map (Fig.2(b)), some representative layouts are obscured by the textual annotations. However, in the zoomed map (c), not only the spatial layout is clearly visible, but more objects are also displayed for MLLM to perform spatial reasoning. Such a zooming operation allows MLLM to capture more spatial clues and to perform fine-grained exploration in local key areas.
固定大小和分辨率的俯视图地图 VMt 可能无法有效地表示空间布局。因此，我们设计了动态地图缩放（DMS）机制，以在探索过程中动态调整地图比例，进一步支持局部的细粒度探索。例如，如原始俯视图地图所示（图2(b)），一些代表性的布局被文本注释遮挡。然而，在缩放后的地图中（c），不仅空间布局清晰可见，而且显示了更多对象，以便MLLM进行空间推理。这种缩放操作使得MLLM能够捕捉更多空间线索，并在局部关键区域进行细粒度探索。
Specifically, we query the MLLM to predict a scaling ratio according to the current top-view map, as shown in Fig.2(b)(c). The MLLM is asked to reason on the original top-view map VMt ,considering the spatial layout and agent's current position etc. Then we require MLLM to select a map scaling ratio r∈[0,1] from a predefined set of candidate scales. According to the scaling ratio chosen by MLLM,the top-view map VMt is cropped with the agent as the center. Then we apply AVPG once again to obtain the scaled top-view map with visual prompts. Note that if r=1 ,which means the top-view map remains unchanged, the original top-view map VMt is directly used for TGN to produce the moving location.
具体来说，我们查询 MLLM 以根据当前的顶视图地图预测缩放比例，如图 2(b)(c) 所示。MLLM 被要求对原始顶视图地图 VMt 进行推理，考虑空间布局和代理的当前位置等。然后，我们要求 MLLM 从预定义的候选缩放比例集中选择一个地图缩放比例 r∈[0,1]。根据 MLLM 选择的缩放比例，顶视图地图 VMt 以代理为中心进行裁剪。然后，我们再次应用 AVPG 以获得带有视觉提示的缩放顶视图地图。请注意，如果 r=1，这意味着顶视图地图保持不变，则直接使用原始顶视图地图 VMt 供 TGN 生成移动位置。

3.6. Local Policy
3.6. 局部策略

Once agent’s decision location (x∗,y∗) is obtained,we calculate a path from the current location to (x∗,y∗) by following previous works [27, 49] (fast marching method [31]). Then a series of low-level actions are generated according to the path by utilizing a deterministic policy. During navigation, if any instance of the goal category is observed, the agent will directly navigate to it. Otherwise, the agent keeps exploring by following MLLM's reasoning results until the maximum number of action steps is reached.
一旦获得代理的决策位置 (x∗,y∗)，我们通过参考先前的工作 [27, 49]（快速推进方法 [31]）计算从当前位置到 (x∗,y∗) 的路径。然后，根据路径生成一系列低级别动作，利用确定性策略。在导航过程中，如果观察到任何目标类别的实例，代理将直接导航到该实例。否则，代理将继续根据 MLLM 的推理结果进行探索，直到达到最大动作步骤数。

5. Conclusion
5. 结论

In this paper, we tackle the Zero-Shot Object Navigation (ZSON) task, where spatial information plays a critical role in such a goal-oriented exploration task. However, previous LLM-based methods must transfer the top-view map to language descriptions, conducting exploration reasoning in the linguistic domain. Such a transformation process loses spatial information such as object and room layout. Therefore, we aim to study how we can directly adopt the top-view map for reasoning by using MLLM's ability to understand images.
