Abstract. Domain generalization (DG) aims to learn a generalized model to an unseen target domain using only limited source domains. Previous attempts to DG fail to learn domain-invariant representations only from the source domains due to the significant domain shifts between training and test domains. Instead, we re-formulate the DG objective using mutual information with the oracle model, a model generalized to any possible domain. We derive a tractable variational lower bound via approximating the oracle model by a pre-trained model, called Mutual Information Regularization with Oracle (MIRO). Our extensive experiments show that MIRO significantly improves the out-of-distribution performance. Furthermore, our scaling experiments show that the larger the scale of the pre-trained model, the greater the performance improvement of MIRO. Code is available at https://github.com/kakaobrain/miro.
摘要。领域泛化（DG）的目标是仅使用有限的源领域来学习一个泛化的模型，以适用于未见过的目标领域。之前的领域泛化尝试由于训练和测试领域之间存在显著的领域偏移，仅从源领域学习领域不变表示失败了。相反，我们使用与先知模型（一种泛化到任何可能领域的模型）的互信息重新构建了DG目标。我们通过用一个预训练模型近似先知模型，导出了一个可操作的变分下界，称为互信息正则化与先知（MIRO）。我们的大量实验表明，MIRO显著提高了分布外性能。此外，我们的规模实验表明，预训练模型的规模越大，MIRO的性能提升越大。代码可在 https://github.com/kakaobrain/miro 找到。

1 Introduction
1 引言

Emerging studies on the generalizability of deep neural networks have revealed that the existing models, which assume independent and identically distributed (i.i.d.) training and test distribution, are not robust to significant distribution shifts between training and test distribution, e.g., backgrounds [62], geographic distribution [59],demographic statistics [52,67] ,textures [3,24] ,or day-to-night shifts [17,42] . Domain generalization (DG) aims to learn robust representations against distribution shifts from multiple source domains during training. The trained model is evaluated on an unseen domain to measure the robustness. The existing DG approaches have tried to learn invariant features across multiple domains [2,11,13,23,33,55,72] . However,recent studies [25,30] have shown that simple baselines without learning invariant features are comparable to or even outperform the existing DG methods on the diverse DG benchmarks with a fair evaluation protocol in realistic settings (e.g., using ResNet-50 instead of ResNet- 18 [26]). We presume that it is because training and test distributions differ too significantly to learn domain-invariant features by the training distribution only.
对深度神经网络泛化性的研究日益显示，现有模型假设训练和测试分布是独立同分布（i.i.d.），却无法对训练和测试分布之间的显著分布偏移具有鲁棒性，例如背景 [62]、地理位置分布 [59]、人口统计 [52,67]、纹理 [3,24] 或昼夜变化 [17,42]。领域泛化（DG）旨在训练期间从多个源领域中学习对分布偏移具有鲁棒性的表征。训练好的模型在未见过的领域中评估其鲁棒性。现有的DG方法试图学习跨多个领域的不变特征 [2,11,13,23,33,55,72]。然而，近期研究 [25,30] 显示，在现实场景下使用公平的评估协议（例如，使用ResNet-50而不是ResNet-18 [26]）时，没有学习不变特征的简单基线方法与现有的DG方法在多样的DG基准测试中性能相当甚至更优。我们推测这是因为训练和测试分布差异过大，仅通过训练分布无法学习到领域不变特征。
Instead of learning domain-invariant features, we let a model learn similar features to "oracle" representations, i.e., an optimal model generalized to any domain. In particular, we re-formulate the DG problem by maximizing the mutual information (MI) between the oracle model representations and the target model representations while preserving the training loss on source domains. However, the oracle model is not achievable in practice. Hence, we use a large pre-trained model (e.g., ImageNet pre-trained ResNet-50 [26]) as an approximation. With this approximation, we derive a tractable variational lower bound of the proposed maximization problem, named Mutual Information Regularization with Oracle (MIRO). At a high level, our MIRO objective consists of two objectives: an original target task (i.e., an ERM objective) and a regularization term between the pre-trained model and the current target model. Note that the standard DomainBed benchmark [25] uses the ImageNet pre-trained ResNet-50 as the initialization of a DG method, thus, we use the pre-trained ResNet as the initialization and the approximation of the oracle model at the same time.
我们没有学习域不变特征，而是让模型学习与“ oracle ”表示相似的特性，即泛化到任何域的最优模型表示。具体来说，我们通过最大化 oracle 模型表示与目标模型表示之间的互信息（MI），同时保持源域上的训练损失，重新构建了域泛化（DG）问题。然而，在实际中无法实现 oracle 模型。因此，我们使用一个大型预训练模型（例如，ImageNet 预训练的 ResNet-50 [26]）作为近似。利用这种近似，我们推导了所提出最大化问题的可操作变分下界，命名为带有 Oracle 的互信息正则化（MIRO）。在较高层面上，我们的 MIRO 目标包括两个目标：原始目标任务（即 ERM 目标）以及预训练模型与当前目标模型之间的正则项。请注意，标准的 DomainBed 基准 [25] 使用 ImageNet 预训练的 ResNet-50 作为域泛化方法的初始化，因此，我们同时使用预训练的 ResNet 作为初始化和 oracle 模型的近似。
While a naive fine-tuning approach of a large pre-trained model can harm the robustness against distribution shifts [32,61] ,our proposed algorithm remarkably improves the robustness against unseen domains during fine-tuning in a plug-and-play manner to any scale of the backbone model and datasets. In our experiment, we observe that the naive fine-tuning of a larger pre-trained model can fail to provide better performances, even though the larger pre-trained model is trained with more data and domains. For example, ERM with the ResNet pre-trained on ImageNet (trained with 1.3M images) shows 64.2% of averaged accuracy, while ERM with the ViT pre-trained on CLIP (trained with 400M image-caption pairs) shows 61.1% . On the other hand,we show that our method can significantly improve the average DG performances with backbone models at different scales,e.g.,ImageNet pre-trained ResNet (64.2%→65.9%) , 400M image-text pre-trained ViT (CLIP) [47] (61.1%→73.7%) and Instagram 3.6B pre-trained RegNet (SWAG) [54] (68.0%→74.1%) . Especially,we observe that the pre-trained knowledge by larger pre-trained models, such as SWAG and CLIP, is more effective to learn domain generalized features than the ImageNet pre-trained model: MIRO with the ViT pre-trained on CLIP outperforms MIRO with the ResNet pre-trained on ImageNet in contrast to the naive fine-tuning. Furthermore, our feature-level regularization method is easily combined with the existing parameter space ensemble methods [13,61](74.1%→77.3% average DG accuracy by combining with SWAD [13] and pre-trained RegNet).
虽然简单地对大型预训练模型进行微调可能会损害对抗分布偏移的鲁棒性 [32,61] ，但我们的提议算法在无需调整的情况下显著提高了在未见过的领域进行微调时的鲁棒性，适用于任何规模的主干模型和数据集。在我们的实验中，我们观察到简单地对更大的预训练模型进行微调可能无法提供更好的性能，尽管更大的预训练模型是在更多的数据和领域上训练的。例如，使用在ImageNet（训练数据为1.3M张图像）上预训练的ResNet的ERM显示出 64.2% 的平均准确度，而使用在CLIP上预训练的ViT（训练数据为 400M 图像-文本对）的ERM显示出 61.1% 。另一方面，我们展示了我们的方法可以显著提高不同规模主干模型的平均域泛化性能，例如，ImageNet预训练的ResNet (64.2%→65.9%) ， 400M 图像-文本预训练的ViT（CLIP）[47] (61.1%→73.7%) 以及Instagram 3.6B预训练的RegNet（SWAG）[54] (68.0%→74.1%) 。特别是，我们观察到，与ImageNet预训练的模型相比，大型预训练模型（如SWAG和CLIP）的预训练知识更有效地学习域泛化特征：使用CLIP预训练的ViT的MIRO超过了使用ImageNet预训练的ResNet的MIRO，与简单微调相比。此外，我们的特征级正则化方法可以轻松地与现有的参数空间集成方法结合 [13,61](74.1%→77.3% ，通过结合SWAD [13] 和预训练的RegNet来提高平均域泛化准确度。
Our contribution is as follows: (1) We re-formulate the DG objective by mutual information with the oracle model. Then, we approximate the oracle by a large pre-trained model to derive a tractable approximation of the target objective. We propose Mutual Information Regularization with Oracle (MIRO) to solve our objective. (2) We analyze the pre-trained models in terms of the MI with the oracle model. Our analysis shows that naive fine-tuning of pre-trained models can harm the MI with the oracle, on the other hand, MIRO shows high MI with the oracle. (3) We compare MIRO with state-of-the-art DG methods on DomainBed. MIRO outperforms all methods in all settings, including varying optimizers and pre-trained models. We also provide extensive analysis to understand MIRO. For example, we observe that MIRO shows stronger DG performances with larger pre-trained models, such as SWAG [54] or CLIP [47].
我们的贡献如下：（1）我们通过使用与Oracle模型的互信息重新构建了DG目标。然后，我们通过一个大型预训练模型来近似Oracle，从而得到目标目标的可处理近似。我们提出了带有Oracle的互信息正则化（MIRO）来解决我们的目标。（2）我们从Oracle模型的角度分析了预训练模型在MI方面的表现。我们的分析表明，对预训练模型进行简单微调可能会损害与Oracle的MI，另一方面，MIRO显示出与Oracle的高MI。（3）我们在DomainBed上比较了MIRO与最先进的DG方法。MIRO在所有设置中均优于所有方法，包括不同的优化器和预训练模型。我们还提供了详尽的分析来理解MIRO。例如，我们观察到MIRO在使用较大的预训练模型，如SWAG [54] 或CLIP [47] 时，显示出更强的DG性能。

2 Related works
2 相关工作

Domain generalization. Learning domain-invariant features from source domains has been a major branch in the DG field. The main idea is discarding biased knowledge to a specific domain while preserving invariant features over source domains, by minimizing feature divergences between the source domains [23,36,38,41,43,55,70] ,simulating domain shifts based on meta-learning [5,11,19,33,35,69] ,robust optimization [2,13,31,51,53] ,or augmenting source domain examples [4,12,44,45,49,66,71,72] . However,even if the model learns invariant representation to source domains, it can still be biased toward the source domains which causes limited performance on unseen target domains. That is, learning invariant representation across source domains is not enough to achieve the underlying objective of domain generalization [11,14,15] . To compensate for the issue, this paper employs pre-trained models, which provide general representations across various domains including unseen target domains.
领域泛化。从源领域中学习域不变特征一直是DG领域的主要分支。主要思想是在保留源域中的不变特征的同时，丢弃对特定域的偏置知识，通过最小化源域之间的特征差异 [23,36,38,41,43,55,70]，基于元学习模拟域转移 [5,11,19,33,35,69]，鲁棒优化 [2,13,31,51,53] 或增强源域示例 [4,12,44,45,49,66,71,72]。然而，即使模型学到了对源域的不变表示，它仍然可能对源域产生偏置，这导致在未见过的目标域上性能受限。也就是说，跨源域学习不变表示并不足以实现领域泛化的基本目标 [11,14,15]。为了弥补这个问题，本文采用了预训练模型，这些模型提供了包括未见过的目标域在内的各种域的通用表示。
Exploiting pre-trained models. There have been numerous attempts to exploit pre-trained models in various fields. Transfer learning [37,64] and knowledge distillation [1,56] employ pre-trained models to improve in-domain performance when dataset or architecture shift occurs between pre-training and fine-tuning. Continual learning utilizes the pre-trained model to maintain old task performance when learning new tasks [40]. Recently, several studies targeting the out-of-distribution generalization are emerging [32,61] . Kumar et al. [32] show that naive fine-tuning distorts the pre-trained features and propose a simple baseline, named LP-FT, to alleviate the distortion. WiSE-FT [61] focuses on zero-shot models. It combines pre-trained and fine-tuned weights to preserve the generalizability of the pre-trained zero-shot models. In this paper, we propose a MI-based regularization method, MIRO, to exploit the generalizability of the pre-trained representation in the training process.
利用预训练模型。在各个领域已有众多尝试来利用预训练模型。迁移学习 [37,64] 和知识蒸馏 [1,56] 使用预训练模型来提高当训练集或架构在预训练和微调之间发生迁移时的域内性能。持续学习利用预训练模型在学习新任务时保持旧任务的性能 [40]。最近，针对分布外泛化的几项研究正在涌现 [32,61]。Kumar等人[32]表明，简单微调会扭曲预训练特征，并提出了一个名为LP-FT的简单基线，以减轻这种扭曲。WiSE-FT [61] 关注于零样本模型。它结合了预训练和微调的权重，以保持预训练零样本模型的泛化性。在本文中，我们提出了一种基于互信息的正则化方法MIRO，以利用预训练表示在训练过程中的泛化性。

3 Methods
3 方法

In this section, we first re-formulate the objective for the out-of-domain generalization by introducing an oracle model. Then, we derive a tractable variational bound of the objective by approximating the oracle model to the pre-trained model. The final form consists of the empirical risk and the mutual information (MI) regularization by querying the approximated oracle, named Mutual Information Regularization with Oracle (MIRO). We empirically validate our approximation by MI between the oracle model and large pre-trained models.
在这一节中，我们首先通过引入一个 oracle 模型来重新构建域外泛化的目标。然后，我们通过将 oracle 模型近似为预训练模型，导出目标的一个可处理的变分界限。最终形式包括经验风险和通过查询近似 oracle 的互信息（MI）正则化，命名为带有 Oracle 的互信息正则化（MIRO）。我们通过查询 oracle 模型与大型预训练模型之间的 MI 来实证验证我们的近似。

Intuitively speaking, the lower bound shows that the smaller d2,∞(f∗,f0) is,the tighter the gap between the true lower bound and approximated one is. 3.3 Features and encoders design
3.3 特征与编码器设计

Multi-scale features. One can only use the last-level features for our regularization. However, high-level features can include pre-training task-related information, often irrelevant to the target task. Instead, we use the intermediate outputs by each model block, i.e., stem output, blocks 1, 2, 3, and 4 for ResNet [26] and RegNet [48], and stem output, blocks 3, 6, 9, and 12 for ViT-B.
多尺度特征。人们只能使用最后一层的特征来进行我们的正则化。然而，高层次特征可能包含与预训练任务相关的信息，这些信息通常与目标任务无关。相反，我们使用每个模型块的中间输出，即茎部输出，ResNet [26] 的块 1、2、3 和 4，以及 RegNet [48] 的茎部输出，块 3、6、9 和 12，对于 ViT-B 则是茎部输出，块 3、6、9 和 12。
Design of the mean and variance encoders. The multi-level structure increases the feature size, resulting in a computational cost increase. We alleviate the issue by employing simple yet effective architectures, identity function for the mean encoder and a bias-only model with diagonal covariance for the variance encoder. We also tested more complicated architectures, but only computational cost was increased without performance improvement.
均值和方差的编码器设计。多层结构增加了特征大小，导致计算成本增加。我们通过采用简单而有效的架构来缓解这个问题，均值编码器使用恒等函数，方差编码器使用只有偏置项和对角协方差模型的架构。我们还测试了更复杂的架构，但仅增加了计算成本，而没有提高性能。

4.3 Analysis of MIRO
4.3 MIRO 分析

Loss function interpretation: ∑ distribution analysis. We can interpret the variance term of MIRO, ∑(zf) in Equation (6),as control variables of the distance loss between pre-trained features zf0 and current learning features zf . During the training phase, if the variance values become smaller then the model will preserve MI with the pre-trained model. On the contrary, when the model needs to learn new information, the variance will increase. We illustrate the learned variances in Figure 2. The figure shows that pre-trained information is preserved well in lower layers, while task-specific new information is learned in higher layers. This result is consistent with the interpretation that high layer features represent more task-specific semantic information than low layer features [20]; task shifts during fine-tuning make higher layer features learn more semantics than lower layers.
损失函数解释：∑ 分布分析。我们可以将 MIRO 的方差项 ∑(zf) 在方程（6）中解释为预训练特征 zf0 和当前学习特征 zf 之间距离损失的控制变量。在训练阶段，如果方差值变小，那么模型将保留与预训练模型的互信息。相反，当模型需要学习新信息时，方差将增加。我们在图 2 中说明了学习的方差。该图显示，在较低层中预训练信息保存得很好，而在较高层中学习到了特定于任务的新信息。这个结果与高层特征比低层特征表示更多特定于任务的语义信息的解释一致 [20]；在微调过程中的任务转换使得高层特征比低层特征学习到更多的语义。

Relationship between the pre-training scale and the intensity of the MI regularization. Our method has a control parameter λ ,which controls the balance between the cross-entropy loss and the MI regularization loss. If λ becomes larger,it implies that the strength of MI regularization becomes stronger, while it weakens the strength of the ERM objective. Intuitively, if the pre-trained knowledge is informative enough to the target task,larger λ will improve the performances, while if the pre-trained knowledge is uninformative to the target task,then larger λ can harm the performances,because of the penalty on the ERM objective. We compare three pre-trained models (ImageNet pre-trained model,SWAG,and CLIP-ViT) by varying λ . 5 Conclusion
5 结论

Traditional domain generalization (DG) approaches focus to learn a robust representation using multiple source domains. However, in the recent trends of scaling up pre-training, the use of a large-scale pre-trained model becomes more important than the use of DG algorithms for the real-world DG. In line with this trend, we propose Mutual Information Regularization with Oracle (MIRO) to robustly exploit the pre-trained model by approximating an oracle model. To do this, we first re-formulate the domain generalization objective by introducing a concept of an oracle model. Then, we derive a tractable variational bound of the objective by approximating the oracle model with the pre-trained model. Our experimental results demonstrate both the effectiveness and the potential of the proposed method. MIRO achieves state-of-the-art performance in the DomainBed benchmarks. Furthermore, when combining MIRO with large-scale pre-trained backbones, such as CLIP [47] or SWAG [54], the performance improvements remarkably increases. We hope that this study promotes a new research direction of exploiting pre-trained backbones to learn robust representations for domain generalization.
传统领域泛化（DG）方法专注于使用多个源域来学习鲁棒表示。然而，在最近大规模预训练的趋势中，大规模预训练模型的使用变得比DG算法对于现实世界DG更为重要。遵循这一趋势，我们提出了带有向导 mutual information 正则化（MIRO），通过近似向导模型来鲁棒地利用预训练模型。为此，我们首先通过引入向导模型的概念来重新构建领域泛化的目标。然后，我们通过用预训练模型近似向导模型，导出了目标的可操作变分界限。我们的实验结果证明了所提出方法的有效性和潜力。MIRO 在 DomainBed 基准测试中实现了最先进的性能。此外，当将 MIRO 与大规模预训练的骨干网络结合使用时，例如 CLIP [47] 或 SWAG [54]，性能提升显著增加。我们希望这项研究促进了利用预训练骨干网络学习用于领域泛化的鲁棒表示的新研究方向。

B. 1 Hyperparameter tuning
B.1 超参数调整

We split the hyperparameters (HPs) into two groups: algorithm-specific HPs and algorithm-agnostic HPs. The algorithm-agnostic HPs consist of batch size, learning rate, dropout, and weight decay, and MIRO has only one algorithm-specific HP,λ . To reduce the computational cost,we tune the algorithm-specific HPs and algorithm-agnostic HPs independently. We first search algorithm-specific HPs with default algorithm-agnostic HPs, then search algorithm-agnostic HPs with the tuned algorithm-specific HPs. That is,the λ is searched in [1.0,0.1 , 0.01,0.001] with the batch size of 32,the learning rate of 5e-5,no dropout,and no weight decay. Then,we search algorithm-agnostic HPs with the searched λ following Cha et al. [13]. They propose reduced HP search space for efficiency compared to DomainBed [25]. The protocol searches the learning rate in [1e-5, 3e-5, 5e-5], dropout in [0.0, 0.1, 0.5], and weight decay in [1e-4, 1e-6]. The batch size per domain is fixed to 32 . Since MIRO is a regularization method, we add a case of no weight decay.
我们将超参数（HPs）分为两组：特定于算法的HPs和独立于算法的HPs。独立于算法的HPs包括批量大小、学习率、dropout和权重衰减，MIRO只有一个特定于算法的 HP,λ。为了降低计算成本，我们独立地调整特定于算法的HPs和独立于算法的HPs。首先，我们使用默认的独立于算法的HPs搜索特定于算法的HPs，然后使用调整后的特定于算法的HPs搜索独立于算法的HPs。也就是说，λ 在 [1.0,0.1 0.01,0.001] 中搜索，批量大小为32，学习率为5e-5，无dropout和无权重衰减。然后，我们根据Cha等人[13]的方法搜索独立于算法的HPs。他们提出了相对于DomainBed [25]的效率降低的超参数搜索空间。该协议在[1e-5, 3e-5, 5e-5]范围内搜索学习率，在[0.0, 0.1, 0.5]范围内搜索dropout，在[1e-4, 1e-6]范围内搜索权重衰减。每个域的批量大小固定为32。由于MIRO是一种正则化方法，我们添加了一个无权重衰减的案例。
Even though we use the efficient HP search protocol, it still requires heavy computational resources. Therefore,we tune λ only for the non-main experiments, including combination with SWAD, combination with various pre-trained backbones, and the case study on Camelyon17. Also, we use the batch size of 16 for SWAG [54] due to the GPU memory limitation. Note that there is room for further performance improvement by intensive HP tuning and additional usage of GPU memory, considering the simplified HP search protocol and limited computational resources.
尽管我们使用了高效的HP搜索协议，但它仍然需要大量的计算资源。因此，我们只为非主要实验调整 λ，包括与SWAD的组合、与各种预训练骨干网络的组合，以及Camelyon17的案例研究。另外，由于GPU内存限制，我们使用SWAG [54] 的批量大小为16。请注意，考虑到简化的HP搜索协议和有限的计算资源，通过密集的HP调整和额外使用GPU内存，还有进一步性能提升的空间。

In general, domain generalization (DG) assumes that there are multiple source domains, source domain labels are available, and the same input has the same label between source and target domains. Here, we can make the variations on the problem settings by changing the assumption. Single-source DG does not assume the multiple source domains [13,21] . Several studies try to solve DG problem without domain labels [12,13,41] . Heterogeneous DG deals with the label set shift, i.e., the same input can have different labels between source and target domains [39,60] . In this task,it is assumed that a classifier is learnable in the target domain and the methods focus on the feature extractor. The proposed method exploits pre-trained models instead of assuming available multiple source domains or source domain labels, and focuses on the feature extractor instead of the classifier. Therefore, MIRO is directly applicable to single-source DG, DG without domain labels, and heterogeneous DG problems. On the other hand, we can consider a more specific type of distribution shift. In this case, we may need a different mutual information (MI) strategy. For example, we can employ class-conditional MI for class-conditional distribution shift (C-MIRO) by using I(Zf∗;Zf∣Y) instead of I(Zf∗;Zf) . In Table 5,C-MIRO achieves comparable scores with MIRO and outperforms ERM even though the problem setting is not class-conditional. From the results, we believe that MIRO can be adapted to other distribution shifts by choosing the proper MI strategy.
通常情况下，领域泛化（DG）假设存在多个源领域，源领域的标签是可用的，并且源领域和目标领域对于相同的输入具有相同的标签。在这里，我们可以通过改变假设来对问题设置进行变化。单源DG不假设存在多个源领域 [13,21] 。一些研究尝试在不使用领域标签的情况下解决DG问题 [12,13,41] 。异质DG处理标签集的转换，即相同的输入在源领域和目标领域可能有不同的标签 [39,60] 。在这个任务中，假设在目标领域中可以学习到分类器，方法集中在特征提取器上。提出的方法利用预训练模型，而不是假设可用的多个源领域或源领域标签，并专注于特征提取器而不是分类器。因此，MIRO可以直接应用于单源DG、无领域标签的DG以及异质DG问题。另一方面，我们可以考虑一种更具体的分布偏移类型。在这种情况下，我们可能需要一个不同的互信息（MI）策略。例如，我们可以通过使用 I(Zf∗;Zf∣Y) 而不是 I(Zf∗;Zf) 来采用类条件MI来解决类条件分布偏移（C-MIRO）。在表5中，尽管问题设置不是类条件的，C-MIRO仍然取得了与MIRO相当的成绩，并且优于ERM。从结果来看，我们相信通过选择适当的MI策略，MIRO可以适应其他分布偏移。
